### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/KonsGio/Admin-UI-Dashboard.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Run start command
   ```js
   npm start
   ```
